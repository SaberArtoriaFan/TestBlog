如何创建一个新Organ

1.创建一个类并继承自OrganBase类,并且设置他的componentType

    public abstract class OrganBase:ComponentBase
    {
        protected abstract ComponentType componentType { get; }
    }

如果是新加的则修改ComponentType枚举值，添加
2.创建一个系统来管理它，即创建一个类并继承自SystemBase，一般来说是**NormalSystemBase**类，并且泛型参数填你创建的OrganBase类

    public class UnitUIShowSystem : NormalSystemBase<UIShowOrgan>

3.决定它什么时候被添加到Entity上，这可能取决于他在初始化时是否引用了其他器官以及被引用，一般来说在XianXia.Unit.UnitMainSystem的SwapnUnitReal方法中偏后的位置添加该器官即可。\
