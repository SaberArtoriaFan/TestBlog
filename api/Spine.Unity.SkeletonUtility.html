flipBy180DegreeRotation

If true, ScaleX and ScaleY are followed by 180 degree rotation. If false, negative Transform scale is used. Note that using negative scale is consistent with previous behaviour (hence the default), however causes serious problems with rigidbodies and physics. Therefore, it is recommended to enable this parameter where possible. When creating hinge chains for a chain of skeleton bones via SkeletonUtilityBone, it is mandatory to have flipBy180DegreeRotation enabled.
