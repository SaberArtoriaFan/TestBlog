Properties

AnimationName

Setting this property sets the animation of the skeleton. If invalid, it will store the animation name for the next time the skeleton is properly initialized. Getting this property gets the name of the currently playing animation. If invalid, it will return the last stored animation name set through this property.

public string AnimationName { get; set; }
