Remarks

This dictionary makes it possible to get the index of a key and a key based on an index. It can be costly to find the index of a key because it must be searched for linearly. It can be costly to insert a key/value pair because other key's indexes must be adjusted. It can be costly to remove a key/value pair because other keys' indexes must be adjusted.
