ComputeWorldVertices(Slot, int, int, float[], int, int)

Transforms the attachment's local Vertices to world coordinates. If the slot's Deform is not empty, it is used to deform the vertices.

See World transforms in the Spine Runtimes Guide. 
public void ComputeWorldVertices(Slot slot, int start, int count, float[] worldVertices, int offset, int stride = 2)

Parameters

slot Slot
start int

The index of the first Vertices value to transform. Each vertex has 2 values, x and y.

count int

The number of world vertex values to output. Must be less than or equal to WorldVerticesLength - start.

worldVertices float[]

The output world vertices. Must have a length greater than or equal to offset + count.

offset int

The worldVertices index to begin writing values.

stride int

The number of worldVertices entries between the value pairs written.
